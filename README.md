# Spotify Tableau Dashboard

This project analyzes Spotify track characteristics using Tableau Desktop.

## Dataset
- Source: Kaggle Spotify Tracks dataset
- File: spotify_tracks.csv

## Dashboard Contents
- Top Artists by Average Popularity
- Track Duration vs Popularity
- Popularity Distribution (Histogram)

## Tools
- Tableau Desktop
- CSV dataset

## Notes
Dashboard built locally to ensure data safety and reproducibility.
